## 🛠️ Getting Started

#### Step 1: 🚀 Initial Setup

- Clone the repository
- Navigate: `cd into repo`
- Install dependencies: `yarn install`

#### Step 2: 🏃‍♂️ Running the Project

- Dev Server: `yarn run dev`
- Build: `yarn run build`
